Welke CSS property kun je gebruiken om ervoor te zorgen dat de waarde van width en height de totale breedte en hoogte is van het blok van de voorgaande opdracht?

block-size: auto;

block size can be used with:

  • Length Values
    block-size: 300px or em
  • Keyword Values
    block-size: max-content, min-content, auto
  • Global Values
    block-size: inherit, intital, revert, unset
  • Percentages
    block-size:75%
