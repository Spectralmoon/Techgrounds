Type Selector

Matches elements by node name

a { }

ID Selector

id attribute of an HTML element to select a specific element.

# a { }

Child Selector

Used to match all the elements which are child of a specified element

a> b { }

Descendant Selector

Made up of two or more selectors separated by white space

a b {}

Combine Descendant & ID

# a b {}

Comma combinator

Separates multiple selectors within the same styles.

a, c {}

Universal

When you wish to put a specific style in all the HTML elements within your web page.

*{}

Combine Universal

Combining the Universal selector with another element

a * {}

Adjacent Sibiling

Separates two selectors and matches the second element only if it immediately follows the first element

a + b{ }

General Sibiling

The CSS general sibling selector is used to select all elements that follow the first element such that both are children of the same parent.

a+b { }

First Child Pseudo

Represents the first element among a group of sibling elements.

b: first-child{ }

Only Child Pseudo

An element that has a parent element and whose parent element has no other element children.

a:only-child {}

Last Child Pseudo

An element that has a parent element and whose parent element has no other element children.

b:last-child { }
